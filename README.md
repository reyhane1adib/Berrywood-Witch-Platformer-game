# Berrywood Witch — 2D Platformer

Berrywood Witch is a whimsical 2D pixel-art platformer created in Unity.  
You play as a small witch exploring a magical forest filled with berries, hazards, and enemies.


---
 Features

- Playable witch character with:
  - Running
  - Jumping
  - Climbing
  - Flash-damage effect
  - Shooting fireballs
  - diffrent animations for idle state, running and climbing
 
 - Berry pickup system  
 -  Enemy collision + lives system  
 -  Parallax forest background  
 -  Custom sound effects  
 -  Level transitions  
 -  Game session + score tracking
 -  3 diffrent levels

How to Play

1. Clone/download the project  
2. Open it in Unity  
3. Load *Main Scene*  
4. Press ▶ Play

-You can use your keyboard and mouse or a controller to play
